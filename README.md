# IT Asset Tracker

A simple and efficient IT Asset Management System built using **React + Firebase**.  
It allows admins to manage company assets, track status, and update records in real-time.

##  Features
-  Admin Login (Firebase Authentication)
-  Add / Update / Delete IT Assets
-  Real-time Data Sync using Firebase Firestore
-  Responsive UI
-  Fast Deployment with Vercel
-  GitHub CI/CD Enabled

##  Tech Stack
- **Frontend:** React (CRA)
- **Database:** Firebase Firestore
- **Auth:** Firebase Authentication
- **Hosting:** Vercel
- **Version Control:** GitHub
